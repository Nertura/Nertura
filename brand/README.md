# Nertura — Logo System

> **Graph Kernel** — enterprise AI operating system mark. Not a farming logo.

**Colors:** `#0B1220` (Void) · `#2DDAAF` (Signal)  
**Status:** Approved vector assets

---

## Symbol Meaning

| Layer | Geometry | Represents |
|-------|----------|------------|
| **Hexagon boundary** | OS kernel | Platform boundary — Palantir-grade precision |
| **Inner triangle mesh** | Three-node graph | Intelligence network — connected inference |
| **Central node** | Filled core | Brain / decision center |
| **Vertex nodes** | Three outer points | Distributed network endpoints |
| **Parallel strata lines** | Two horizontal strokes | Data layers + subtle cultivated-row geometry (agriculture as structure, not biology) |

No leaves. No tractors. No clipart. No gradients.

---

## Clear Space

Minimum padding around mark = height of central node (×) on all sides.

## Minimum Size

| Context | Min |
|---------|-----|
| Symbol digital | 16px |
| Symbol print | 6mm |
| Wordmark cap height | 12px digital |

## Misuse

- Do not add gradients, shadows, or glow
- Do not recolor outside Void / Signal / white
- Do not rotate or stretch
- Do not replace triangle with leaf shapes
- Do not use on busy photography without Void scrim
