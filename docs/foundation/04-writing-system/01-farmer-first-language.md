# 01 — Farmer-First Language

## Purpose

Define how Nertura speaks to farmers — in Turkey and globally.

---

## Principles

### 1. Farmer-first, never engineer-first

Write for a person in the field, not a developer reading logs.

| Bad | Good (TR) |
|-----|-----------|
| "Upload failed: 413" | "Fotoğraf çok büyük. 5 MB altında bir görsel seçin." |
| "Insufficient credits" | "Analiz hakkınız kalmadı." |
| "Retry" | "Tekrar dene" |

### 2. Natural Turkish — never "translated English"

Turkish copy must be authored for Turkish farmers, not translated from English source strings.

### 3. One language per flow

No mixed TR/EN in a single user journey. See Book 04 Ch. 07.

### 4. Short-first for Doctor answers

Lead with actionable guidance. Details follow. See Book 04 Ch. 02.

### 5. Respect uncertainty

Agricultural advice is probabilistic. Language must reflect that.

| Bad | Good (TR) |
|-----|-----------|
| "This is tomato blight." | "Olası durum: Domates mildiyosu." |
| "Apply fungicide immediately." | "Bugün ne yapın: … Yerel uzmanınıza danışın." |

### 6. High-impact advice requires care

Chemicals, pesticides, dosage, safety — always include expert verification language when risk is high.

---

## Approved Product Terms (TR)

Use consistently:

| Term | Usage |
|------|-------|
| **analiz hakkı** | Usage limits — never "credits" farmer-facing |
| **AI Tarım Doktoru** | Product name in farmer context |
| **Vaka Takibi** | Case tracking — never "case" or "project" TR-visible |
| **Tarla Profili** | Field context |
| **Bilgi Bankası** | Knowledge source label |
| **Kanıt Kartları** | Evidence panel |
| **Olası durum** | Diagnosis header — not "Diagnosis" |
| **Bugün ne yapın** | Immediate action — not "Treatment plan" |
| **Takip önerisi** | Follow-up suggestion |
| **Güven** | Confidence label — not "Confidence" |

---

## Voice Attributes

| Attribute | Expression |
|-----------|------------|
| **Grounded** | Specific crops, regions, seasons — not generic "your plant" |
| **Calm** | No alarm unless severity warrants |
| **Respectful** | Farmer knows their land; Nertura advises |
| **Honest** | Admit limits; suggest clearer photo or expert |
