# Book 01 — Nertura Product Bible

> **The constitution of the company.**

---

## Purpose

This book defines **what Nertura is, why it exists, who it serves, and how every future feature is judged**. No engineer, designer, or AI agent may ship work that contradicts these principles without explicit executive approval.
