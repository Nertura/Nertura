# 02 — Surface Feel Map

## Purpose

Map every major Nertura surface to its intended **emotional outcome** and **primary user goal**.

---

## Farmer-Facing Surfaces

| Surface | Primary feel | User goal | Must NOT feel like |
|---------|--------------|-----------|-------------------|
| **Landing (Marketing `/`)** | Trust, clarity, calm | "Can this help my crop?" | SaaS landing hype |
| **AI Doctor** | Guidance, expertise, care | "What should I do today?" | Generic chatbot |
| **Vaka Takibi (`/cases`)** | Progress, continuity, memory | "What's happening with this problem?" | Project management tool |
| **Case detail** | Clarity, timeline, confidence | "What happened and what's next?" | Medical chart jargon |
| **Tarlalar / Farms** | Ownership, control, digital farm | "This is my land" | GIS software |
| **Field workspace** | Context, history, action | "Everything about this field" | Spreadsheet |
| **History** | Memory, not archive | "What did we discuss before?" | Server logs |
| **Settings** | Control, privacy, confidence | "I'm in charge of my data" | Developer console |
| **Account / Billing** | Transparency, no pressure | "What do I get and what does it cost?" | Aggressive upsell |
| **Onboarding** | Welcome, progress, achievable | "Set up takes minutes" | Enterprise wizard |
| **Auth (login/register)** | Safe, simple, respectful | "Sign in without friction" | Identity bureaucracy |

---

## Internal / Growth Surfaces

| Surface | Primary feel | User goal | Must NOT feel like |
|---------|--------------|-----------|-------------------|
| **Admin** | Power, precision, safety | "Operate the platform responsibly" | Hacker dashboard |
| **Growth AI (admin)** | Approval-based, never spam | "Draft → review → send" | Autopilot marketing |
| **Knowledge (admin)** | Evidence, review, scientific caution | "Curate trusted knowledge" | Wiki free-for-all |

---

## Experience Hierarchy (Doctor)

Per Book 02 Ch. 07 and Nertura Core:

1. **Conversation** — composer, messages
2. **Evidence** — Kanıt Kartları, Bilgi Bankası
3. **Actions** — Bugün ne yapın, Takip önerisi
4. **History** — past conversations
5. **Cases** — Vaka Takibi linkage

Never invert this order visually.

---

## Mobile Feel Additions

| Constraint | Rule |
|------------|------|
| Touch | Minimum 44×44px targets |
| Scroll | No horizontal scroll on primary flows |
| Drawers | Bottom sheets for actions; not full-page traps |
| Outdoor use | High contrast body text; readable in sunlight |

---

## Review Checklist (Per Screen)

- [ ] Primary feel matches this map
- [ ] User goal achievable in one obvious action
- [ ] No competing emotional signals (calm + alarm banner overload)
- [ ] Turkish farmer would describe screen in one plain sentence
